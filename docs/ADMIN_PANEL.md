# دليل لوحة التحكم — Edurim Admin Panel

## تسجيل الدخول

1. شغّل Backend: `cd backend && go run cmd/server/main.go`
2. شغّل Admin Panel: `cd admin && flutter run -d chrome`
3. أدخل بيانات المسؤول التجريبي:
   - **البريد:** `admin@edurim.local`
   - **كلمة المرور:** `admin12345`

---

## لوحة الإحصائيات (Dashboard)

تعرض عند الدخول:
- عدد الطلاب المسجلين
- عدد المواد الدراسية
- عدد الدروس
- عدد التمارين
- عدد مواضيع الامتحانات النشطة

---

## إضافة مادة دراسية

1. انقر **المواد** في الشريط الجانبي
2. انقر **إضافة مادة**
3. املأ الحقول:
   - **اسم المادة** (بالعربية) — مثال: الرياضيات
   - **الأيقونة** (Emoji) — مثال: 📐
   - **اللون** (hex) — مثال: `#2563EB`
   - **رابط صورة الغلاف** (اختياري) — مثال: `/uploads/covers/math.jpg`
   - **المسار الدراسي** — كونكور / BEPC / باكالوريا
4. انقر **حفظ**

المادة تظهر فوراً في تطبيق الطالب عند تحديث الصفحة.

---

## تعديل مادة

1. في قائمة المواد، انقر أيقونة القلم ✏️ بجانب المادة
2. عدّل الحقول المطلوبة
3. انقر **حفظ**

---

## إضافة درس

1. انقر **الدروس** في الشريط الجانبي
2. انقر **إضافة**
3. أدخل:
   - **عنوان الدرس**
   - **الوصف**
   - **رابط الفيديو** — YouTube أو أي رابط خارجي
   - **رابط الملخص PDF** — رابط مباشر للملف
   - **رابط صورة الغلاف** (اختياري)
   - **ID المادة** — رقم المادة من قاعدة البيانات
   - **ID الأستاذ** — 0 إذا لم يكن هناك أستاذ محدد
   - **مجاني / مدفوع** — Toggle
4. انقر **حفظ**

---

## إضافة تمرين

1. انقر **التمارين** في الشريط الجانبي
2. انقر **إضافة**
3. أدخل:
   - **العنوان**
   - **ID المادة**
   - **السنة** — مثال: 2023
   - **الصعوبة** — سهل / متوسط / صعب
   - **رابط ملف التمرين** — PDF أو رابط خارجي
   - **رابط ملف الحل**
   - **رابط فيديو الحل** (اختياري)
   - **رابط صورة الغلاف** (اختياري)
4. انقر **حفظ**

---

## رفع صورة أو ملف PDF

### عبر API مباشرة (curl أو Postman):

```bash
# احصل على token أولاً من تسجيل دخول المسؤول
TOKEN="eyJ..."

# رفع صورة غلاف
curl -X POST "http://localhost:8081/api/admin/uploads?type=covers" \
  -H "Authorization: Bearer $TOKEN" \
  -F "file=@/path/to/image.jpg"

# رفع ملف PDF
curl -X POST "http://localhost:8081/api/admin/uploads?type=files" \
  -H "Authorization: Bearer $TOKEN" \
  -F "file=@/path/to/document.pdf"
```

### الاستجابة:
```json
{ "url": "/uploads/covers/1234567890.jpg" }
```

استخدم هذا الـ URL مباشرة في حقول صورة الغلاف أو روابط الملفات.

### أنواع الرفع:
| type | المجلد | الاستخدام |
|------|--------|----------|
| `covers` | uploads/covers/ | صور الغلاف |
| `files` | uploads/files/ | ملفات PDF والمستندات |
| `images` | uploads/images/ | صور عامة |

**الأنواع المسموحة:** jpg, jpeg, png, webp, pdf, mp4
**الحجم الأقصى:** 20MB

## إضافة موضوع امتحان سابق

1. انقر **مواضيع الامتحانات** في الشريط الجانبي
2. انقر **إضافة**
3. أدخل:
   - **العنوان** — مثال: موضوع BEPC 2023 - الرياضيات
   - **السنة** — مثال: 2023
   - **ID المادة**
   - **الوصف** (اختياري)
   - **رابط ملف الموضوع** — PDF
   - **رابط ملف الحل** — PDF (اختياري)
   - **رابط صورة الغلاف** (اختياري)
   - **المسار** — كونكور / BEPC / باكالوريا
   - **الشعبة** — تظهر فقط إذا اخترت باكالوريا
4. انقر **حفظ**

---

## التحقق من ظهور المحتوى في تطبيق الطالب

1. افتح تطبيق الطالب `cd frontend && flutter run -d chrome`
2. سجّل دخول بأي حساب طالب
3. إذا كانت هذه أول مرة: اختر المسار الدراسي
4. في الصفحة الرئيسية ستظهر مواد مسارك
5. افتح مادة → ستجد الدروس، التمارين، والامتحانات السابقة

> **ملاحظة:** التطبيق يجلب البيانات من API مباشرة عند كل تحميل، لا يوجد cache ثابت. المحتوى يظهر فوراً بعد الإضافة من Admin.

## إدارة المسؤولين

حالياً يوجد مسؤول واحد تجريبي. لإضافة مسؤول جديد:

```sql
-- احسب bcrypt hash لكلمة المرور أولاً
INSERT INTO admins (full_name, email, password_hash, role, is_active)
VALUES ('اسم المسؤول', 'email@example.com', '$2a$10$...hash...', 'content_manager', 1);
```

**الأدوار:**
- `super_admin` — صلاحيات كاملة
- `content_manager` — إدارة المحتوى فقط
